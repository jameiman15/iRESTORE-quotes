# WellMike 報價單管理系統

## 部署步驟

### 1. 上傳到 GitHub
1. 前往 github.com → New repository → 名稱 `wellmike-quotes` → Create
2. 點 "uploading an existing file"
3. 把這個資料夾內的所有檔案拖曳上去
4. 點 "Commit changes"

### 2. 串接 Vercel
1. 前往 vercel.com → Add New Project
2. Import 你的 `wellmike-quotes` repo
3. Framework Preset 選 "Other"
4. 點 Deploy

### 3. 完成
取得網址如：https://wellmike-quotes.vercel.app
